## 📊 自动化配置详情

### project.config.json 配置
```json
{
  "appid": "touristappid",
  "setting": {
    "urlCheck": false,        // 不校验合法域名
    "es6": true,              // 启用ES6转ES5
    "enhance": true,          // 增强编译
    "minified": true,         // 代码压缩
    "minifyWXML": true,       // WXML压缩
    "minifyWXSS": true        // WXSS压缩
  }
}
```

### app.js 配置
```javascript
globalData: {
  baseUrl: 'http://localhost:3000/api',  // 本地服务器地址
  socketUrl: 'ws://localhost:3000'         // WebSocket地址
}
```
