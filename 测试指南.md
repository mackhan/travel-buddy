## ❌ 异常情况处理

### 问题1：仍然显示云开发错误
**原因：** 编译缓存问题
**解决：**
1. 点击"清缓存" → "清除文件缓存"
2. 点击"清缓存" → "清除全部缓存"
3. 重新编译

### 问题2：页面空白
**原因：** 配置未更新
**解决：**
1. 关闭微信开发者工具
2. 重新打开
3. 重新导入项目

### 问题3：无法连接服务器
**说明：** 这是正常的！
- 系统会自动切换到 Mock 模式
- 所有功能都可以正常使用
- 不影响体验

## 🔍 调试技巧

### 查看网络请求
1. 打开调试器
2. 切换到 "Network" 标签
3. 查看 `/api/health` 请求
4. 状态码应该是 200

### 查看全局数据
在 Console 中输入：
```javascript
console.log(getApp().globalData)
```

查看：
- baseUrl: "http://localhost:3000/api"
- mockMode: false（或 true）

### 手动测试后端连接
在 Console 中输入：
```javascript
wx.request({
  url: 'http://localhost:3000/api/health',
  success: (res) => console.log('后端响应:', res),
  fail: (err) => console.error('连接失败:', err)
})
```
