## 📋 完整操作步骤

### 第1步：开启 CLI 服务端口（关键！）

**为什么需要：** 
- 如果不开启，CLI 命令无法控制微信开发者工具
- 这是使用自动化工具的前提

**操作步骤：**
1. 打开微信开发者工具
2. 点击右上角 **"设置"** 按钮（齿轮图标）
3. 在弹出的窗口中选择 **"通用设置"** 标签
4. 滚动找到 **"安全设置"** 部分
5. ✅ **勾选 "开启服务端口"**
6. 点击"确定"按钮保存

**示意图位置：**
```
微信开发者工具界面
├── 右上角
│   └── 设置 (齿轮图标) → 点击
│       └── 通用设置 → 点击
│           └── 安全设置
│               └── ✅ 开启服务端口 (勾选这个！)
```

---

### 第2步：关闭当前项目

**操作步骤：**
1. 在微信开发者工具中，点击左上角的项目名称
2. 在下拉菜单中选择 **"关闭项目"**
3. 或者直接点击项目窗口的关闭按钮

### 第3步：重新导入项目（最关键！）

**操作步骤：**

#### 3.1 点击导入
- 点击窗口中间的 **"+"** 号按钮
- 或者点击 **"导入项目"** 按钮

#### 3.2 选择项目目录
- 点击 **"选择目录"** 按钮
- 浏览到：`c:\Users\nierhan\CodeBuddy\Claw\travel-buddy\client`
- **重要：** 确保选中 `client` 文件夹，不是上级目录

#### 3.3 填写项目信息

**项目名称：**
```
Travel Buddy
```

**AppID（最重要的一步！）：**
```

⚠️ 不要手动输入 "touristappid"
⚠️ 不要输入任何其他字符串
✅ 只从下拉框中选择 "测试号"
```

**开发模式：**
```
小程序
```

**后端服务：**
```
不使用云服务
```

#### 3.4 确认导入
- 检查所有信息是否正确
- 点击 **"导入"** 或 **"确定"** 按钮
- 等待项目加载完成（可能需要几秒钟）

---

### 第4步：配置本地设置（必须！）

**操作步骤：**

#### 4.1 打开项目设置
- 点击窗口右上角的 **"详情"** 按钮

#### 4.2 切换到本地设置
- 在弹出的窗口中，点击 **"本地设置"** 标签

#### 4.3 勾选必要选项（全部勾选！）

**必须勾选的选项：**
```
✅ 不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书
✅ 启用 ES6 转 ES5
✅ 增强编译
✅ 不校验 web-view（业务域名）
```

**这些选项的作用：**
- **不校验合法域名：** 允许访问 localhost（本地服务器）
- **启用 ES6 转 ES5：** 转换现代 JavaScript 语法
- **增强编译：** 提供更好的编译优化
- **不校验 web-view：** 允许嵌入网页

#### 4.4 保存设置
- 点击 **"确定"** 按钮保存配置

---

## ❌ 常见错误与解决

### 错误1：AppID 错误提示
**错误信息：** `touristappid 无效`

**原因：** 
- 手动输入了 `touristappid`
- 应该从下拉框选择"测试号"

**解决方法：**
1. 重新导入项目
2. 在 AppID 下拉框中选择 **"测试号"**
3. 不要手动输入任何内容

---

### 错误2：CLI 服务端口未开启
**错误信息：** `CLI服务端口未开启`

**原因：**
- 微信开发者工具没有开启服务端口
- CLI 命令无法控制工具

**解决方法：**
1. 点击 设置 → 通用设置 → 安全设置
2. 勾选 "开启服务端口"
3. 点击"确定"
4. 重新执行 CLI 命令

---

### 错误3：网络错误
**错误信息：** `request:fail url not in domain list`

**原因：**
- URL 校验未关闭
- 无法访问 localhost

**解决方法：**
1. 点击 "详情" → "本地设置"
2. 勾选 "不校验合法域名"
3. 点击 "清缓存" → "清除全部缓存"
4. 重新编译

---

### 错误4：小程序空白
**错误信息：** 预览区显示空白

**原因：**
- 代码编译失败
- 配置错误

**解决方法：**
1. 查看控制台错误信息
2. 点击 "清缓存" → "清除文件缓存"
3. 检查 app.json 中的 pages 配置
4. 重新编译

---

### 错误5：无法连接后端
**错误信息：** `无法连接服务器`

**原因：**
- 后端服务未启动
- URL 配置错误

**解决方法：**
1. 确认后端服务是否运行（端口3000）
2. 检查 app.js 中的 baseUrl 配置
3. 这是正常的！会自动切换到 Mock 模式
4. 所有功能都可以正常使用

### 自动编译
- 修改代码后，保存文件
- 微信开发者工具会自动重新编译
- 预览区会自动刷新

### 手动编译
