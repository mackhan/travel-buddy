## 🔧 常见问题解决

### 问题1：无法连接服务器

**现象**：提示"⚠️ 无法连接服务器"

**原因**：后端服务未运行

**解决**：
```bash
cd c:/Users/nierhan/CodeBuddy/Claw/travel-buddy/server
npm run dev
```

### 问题2：提示网络错误

**现象**：控制台显示"request:fail"

**原因**：未勾选"不校验合法域名"

**解决**：
1. 点击"详情" → "本地设置"
2. 勾选"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"
3. 重新编译

### 问题3：小程序空白

**原因**：编译错误

**解决**：
1. 点击"清缓存" → "清除全部缓存"
2. 重新编译
3. 查看控制台错误信息

### 问题4：无法显示图片

**原因**：图片路径错误或未勾选配置

**解决**：
1. 检查"本地设置"是否正确勾选
2. 清除缓存后重新编译

### 问题5：Mock模式说明

**现象**：提示"使用演示数据"

**说明**：这是正常现象！
- 当无法连接真实服务器时，会自动使用模拟数据
- 不影响功能体验
- 所有功能都可以正常使用
