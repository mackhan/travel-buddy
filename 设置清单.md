# 微信开发者工具配置脚本

## 设置项说明

需要在微信开发者工具中手动完成以下设置：

### 1. 设置 → 通用设置 → 安全设置
- [ ] 勾选 "开启服务端口"
- [ ] 点击"确定"保存

### 2. 导入项目步骤
1. 关闭当前项目
2. 点击 "+" 号或 "导入项目"
3. 选择目录：c:/Users/nierhan/CodeBuddy/Claw/travel-buddy/client
4. AppID 选择：**测试号**（不要手动输入 touristappid）
5. 项目名称：Travel Buddy
6. 开发模式：小程序
7. 后端服务：不使用云服务
8. 点击"导入"

### 3. 项目设置 → 本地设置
- [ ] 不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书
- [ ] 启用 ES6 转 ES5
- [ ] 增强编译
- [ ] 不校验 web-view（业务域名）

### 4. 编译运行
1. 点击 "编译" 按钮
2. 等待编译完成
3. 查看左侧预览区
