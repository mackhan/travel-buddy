## 📱 前端小程序编译步骤

### 第一步：下载微信开发者工具
1. 访问：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html
2. 根据你的操作系统下载对应版本（Windows/Mac）
3. 安装并启动微信开发者工具

### 第二步：扫码登录
1. 打开微信开发者工具后，使用微信扫码登录

### 第三步：导入项目
1. 点击"+"号或"导入项目"按钮
2. 选择项目目录：
   ```
   c:/Users/nierhan/CodeBuddy/Claw/travel-buddy/client
   ```
3. 填写项目信息：
   - **AppID**：选择"测试号"（开发测试用）
   - **项目名称**：Travel Buddy（或自定义）
   - **开发目录**：已自动填入上方的目录路径
4. 点击"导入"

### 第四步：配置开发环境（重要）
导入项目后，需要配置才能访问本地服务器：

1. 点击右上角"详情"按钮
2. 在"本地设置"标签页中：
   - ✅ 勾选 **"不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书"**
   - ✅ 勾选 **"启用 ES6 转 ES5"**
   - ✅ 勾选 **"增强编译"**
3. 保存设置

### 第五步：编译运行
1. 点击顶部菜单的"编译"按钮（或按 Ctrl+B）
2. 等待编译完成，即可在左侧预览区看到小程序界面

## 🚀 常见问题

### Q1: 提示"网络错误"或"连接服务器失败"？
**A**：确保后端服务正在运行：
```bash
cd c:/Users/nierhan/CodeBuddy/Claw/travel-buddy/server
npm run dev
```

### Q2: 小程序空白或报错？
**A**：
1. 检查是否勾选了"不校验合法域名"
2. 点击"清缓存" → "清除全部缓存"
3. 重新编译

### Q3: 如何查看控制台日志？
**A**：点击底部菜单的"调试器" → "Console" 标签页

### Q4: Mock 模式是什么？
**A**：Mock 模式是模拟数据模式，当无法连接真实服务器时自动启用，所有功能使用本地模拟数据，不影响体验。

### Q5: 能否在手机上预览？
**A**：
1. 点击"预览"按钮生成二维码
2. 使用微信扫码即可在手机上查看
3. 注意：手机预览无法访问本地服务器，会自动使用 Mock 模式
