# 搭牛牛 - 旅行搭子匹配平台

## 项目简介

搭牛牛是一个非盈利的微信小程序，旨在帮助世界各地旅行的人找到志同道合的搭子。用户可以筛选固定日期和城市的信息，找到同城的人，并询问是否可以一起出发。

### 核心功能

- 找搭子：根据城市、日期、活动类型筛选搭子
- 发搭子：发布自己的旅行计划，寻找伙伴
- 搭子详情：查看搭子详细信息并发起邀请
- 个人中心：管理个人信息和统计数据

### 设计风格

参考"粗门"小程序的年轻化设计风格：
- 主色调：粉色到橙色渐变（#FF6B9D → #FF8E53）
- 圆角卡片设计（24rpx）
- 柔和的渐变背景
- 活力图标和清晰的层次结构

## 功能说明

### 1. 首页（找搭子）

**文件位置**：`pages/index/index.js`

**主要功能**：
- 城市筛选：选择目的地城市
- 日期筛选：选择出发日期
- 活动类型标签：美食、观光、购物、摄影、探险
- 搜索功能：支持关键词搜索
- 搭子列表：展示附近搭子信息
- 跳转详情：点击搭子卡片查看详情

**核心方法**：
- `filterPartners()`：筛选搭子的核心方法
- `getLocation()`：获取用户当前位置
- `goToDetail()`：跳转到搭子详情页面

### 2. 发搭子页面

**文件位置**：`pages/publish/publish.js`

**主要功能**：
- 城市选择：自动定位或手动选择
- 日期选择：选择出发日期
- 时间选择：选择出发时间
- 活动类型：多选标签
- 计划描述：详细说明旅行计划
- 人数选择：1-10人可选
- 费用说明：AA制、我请客、面议
- 表单验证：必填项检查
- 发布成功：跳转回首页

**核心方法**：
- `toggleTag()`：切换活动类型标签
- `submitForm()`：提交表单并验证
- `decreaseCount()`/`increaseCount()`：调整需要人数

### 3. 搭子详情页面

**文件位置**：`pages/detail/detail.js`

**主要功能**：
- 用户信息展示：头像、昵称、城市
- 计划详情展示：日期、时间、人数、费用
- 活动类型标签：显示活动类型
- 发起留言：填写留言内容
- 发起邀请：发送邀请给搭子
- 分享功能：分享给朋友

**核心方法**：
- `loadPartnerDetail()`：加载搭子详细信息
- `sendRequest()`：发起邀请
- `sharePartner()`：分享搭子信息

### 4. 个人中心页面

**文件位置**：`pages/my/my.js`

**主要功能**：
- 用户信息展示：头像、昵称、ID
- 统计数据：发布、参与、粉丝
- 功能菜单：我的搭子、消息通知、收藏、历史记录
- 编辑资料：更新用户信息
- 关于我们：显示项目介绍
- 联系客服：显示客服信息

**核心方法**：
- `loadUserInfo()`：加载用户信息
- `editProfile()`：编辑个人资料
- `generateUserId()`：生成用户ID

## 注意事项

### 数据存储

当前项目使用模拟数据，实际使用时需要：

1. **接入后端 API**
   - 搭建后端服务器
   - 设计数据库表结构
   - 开发 RESTful API 接口
   - 替换模拟数据为 API 调用

2. **用户认证**
   - 实现微信登录功能
   - 获取用户 openid
   - 管理用户会话

3. **数据同步**
   - 搭子数据需要从服务器获取
   - 用户信息需要保存到服务器
   - 实现实时数据更新

### 地图服务

- 腾讯地图 API Key 需要替换为真实的 key
- 地图服务需要配置域名白名单
- 注意 API 调用次数限制

### 用户隐私

- 遵守微信小程序隐私规范
- 获取用户位置需要用户授权
- 获取用户信息需要用户授权
- 不要收集不必要的用户数据

### 安全建议

- 对用户输入进行验证和过滤
- 使用 HTTPS 协议进行网络请求
- 不要在代码中硬编码敏感信息
- 定期更新依赖库

## 常见问题

### Q: 如何修改小程序名称？

A: 在微信公众平台登录后，进入"设置" → "基本设置" → 修改小程序名称。

### Q: 如何修改小程序图标？

A: 在微信公众平台登录后，进入"设置" → "基本设置" → 上传小程序图标。

### Q: 如何添加新的页面？

A: 
1. 在 `pages` 目录下创建新的页面文件夹
2. 创建 `.wxml`、`.js`、`.wxss`、`.json` 四个文件
3. 在 `app.json` 的 `pages` 数组中添加页面路径

### Q: 如何调试网络请求？

A: 在微信开发者工具中，打开"调试器" → "Network" 标签，查看网络请求详情。

### Q: 如何处理用户拒绝授权？

A: 在代码中添加授权失败的回调处理，引导用户到设置页面开启权限。
