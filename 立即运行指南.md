## 📱 微信开发者工具操作步骤

### 步骤 2：扫码登录
- 使用微信扫描屏幕上的二维码
- 扫码后会自动登录

### 步骤 3：导入项目（关键步骤）

#### 3.1 点击导入
- 点击窗口右上角的 **"+"** 号
- 或点击"导入项目"按钮

#### 3.2 选择项目目录
- 点击"选择目录"按钮
- 浏览到：`c:/Users/nierhan/CodeBuddy/Claw/travel-buddy/client`
- 确认选中 `client` 文件夹

#### 3.3 填写项目信息
```
项目名称：Travel Buddy
目录：c:/Users/nierhan/CodeBuddy/Claw/travel-buddy/client
AppID：选择"测试号"
开发模式：小程序
后端服务：不使用云服务
```

#### 3.4 确认导入
- 点击"导入"或"确定"按钮
- 等待项目加载完成

---

### 步骤 4：配置开发环境（⚠️ 非常重要！）

#### 4.1 打开项目设置
- 点击窗口右上角的 **"详情"** 按钮

#### 4.2 配置本地设置
在弹出的窗口中，切换到 **"本地设置"** 标签页，然后：

**必须勾选以下选项：**

```
✅ 不校验合法域名、web-view（业务域名）、TLS 版本以及 HTTPS 证书
✅ 启用 ES6 转 ES5
✅ 增强编译
✅ 不校验 web-view（业务域名）
```

#### 4.3 保存设置
- 点击"确定"按钮保存配置

**⚠️ 警告：不勾选这些选项会导致小程序无法连接本地服务器！**

---

#### 5.1 开始编译
- 点击顶部工具栏的 **"编译"** 按钮

#### 5.2 观察编译过程
- 底部状态栏会显示编译进度

- 显示：`✅ 服务器已连接`
- 或显示：`⚠️ 无法连接服务器，使用演示数据`（这也是正常的）

**状态栏：**

## 🔧 常见问题快速解决

### 问题1：提示"网络错误"
**解决方法：**
1. 点击"详情" → "本地设置"
2. 勾选"不校验合法域名"
3. 点击"清缓存" → "清除全部缓存"
4. 重新编译

### 问题2：小程序空白
**解决方法：**
1. 点击"清缓存" → "清除文件缓存"
2. 重新编译
3. 查看控制台错误信息

### 问题3：无法显示图片
**解决方法：**
1. 确认已勾选所有"本地设置"选项
2. 清除缓存后重新编译

### 问题4：无法连接服务器
**解决方法：**
- 这是正常的！小程序会自动切换到Mock模式
- 所有功能都可以正常使用
